# BondUniverse - Apple TV App

This is a native tvOS application built with SwiftUI, featuring a premium "OS 26" aesthetic.

## How to Run

1.  **Install Xcode**: Ensure you have Xcode installed from the Mac App Store.
2.  **Create Project**:
    *   Open Xcode.
    *   Select "Create a new Xcode project".
    *   Choose **tvOS** -> **App**.
    *   Product Name: `BondUniverse`.
    *   Interface: **SwiftUI**.
    *   Language: **Swift**.
    *   Save it to a temporary location or your desktop.
3.  **Import Files**:
    *   Navigate to the folder where you created the project.
    *   **Replace** the contents of the `BondUniverse` folder (the one containing `ContentView.swift`, etc.) with the files in `/Users/shubh/dev/BondUniverse`.
    *   Alternatively, drag and drop the files from `/Users/shubh/dev/BondUniverse` into the Xcode project navigator, checking "Copy items if needed".
4.  **Run**:
    *   Select the **Apple TV 4K** simulator from the top bar.
    *   Press **Cmd + R** to build and run.

## Features

*   **Universe Navigation**: Switch between different cinematic universes (Bond, Marvel, Star Wars, etc.).
*   **Chronological Viewing**: See movies in their correct viewing order.
*   **Premium UI**: Glassmorphism, ambient lighting, and smooth focus animations optimized for the Siri Remote.
